Package com.nuecho.rivr.voicexml.turn.last

VoiceXML last turn classes.

See: Description

  • Class Summary 
    Class Description
    Disconnect
    Terminates a VoiceXmlDialogue with a disconnect.
    Exit
    Terminates a VoiceXmlDialogue with an exit, returning the control to the platform.
    Goto
    Terminates a VoiceXmlDialogue with a goto, transferring the control to another application specified with a URI.
    Return
    Terminates a VoiceXmlDialogue with a return, transferring the control back to the calling dialogue.
    Submit
    Terminates a VoiceXmlDialogue with a submit, sending information to the server and leaving control to the document returned by the request.
    VoiceXmlLastTurn
    Base class for the last turn returned by a VoiceXmlDialogue.
