    public class Submit
extends VoiceXmlLastTurn
    Terminates a VoiceXmlDialogue with a submit, sending information to the server and leaving control to the document returned by the request. Submit can be performed with a GET or a POST.
    See Also:
    VoiceXmlDialogue, LastTurn, SubmitMethod, http://www.w3.org/TR/voicexml20/#dml5.3.8
