Package com.nuecho.rivr.voicexml.turn.output

VoiceXML output turn classes.

See: Description

  • Class Summary 
    Class Description
    BlindTransfer
    A BlindTransfer is a Transfer where the outcome is not monitored.
    BlindTransfer.Builder
    Builder used to ease the creation of instances of BlindTransfer.
    BridgeTransfer
    A BridgeTransfer is a SupervisedTransfer that connects the caller to the callee in a full duplex conversation.
    BridgeTransfer.Builder
    Builder used to ease the creation of instances of BridgeTransfer.
    ConsultationTransfer
    A ConsultationTransfer is a is a SupervisedTransfer that is similar to a BlindTransfer except that the outcome of the transfer call setup is known and the caller is not dropped as a result of an unsuccessful transfer attempt.
    ConsultationTransfer.Builder
    Builder used to ease the creation of instances of ConsultationTransfer.
    DtmfRecognition
    A DtmfRecognition represents the DTMF part of an interaction phase recognition configuration.
    Interaction
    An Interaction is a VoiceXmlOutputTurn that represents a list of Interaction.Prompt with an optional Interaction.FinalRecognitionWindow or Interaction.FinalRecordingWindow phase.
    Interaction.Builder
    Builder used to ease the creation of instances of Interaction.
    Interaction.FinalRecognitionWindow
    A Interaction.FinalRecognitionWindow is an optional final phase of an Interaction.
    Interaction.FinalRecordingWindow
    A Interaction.FinalRecordingWindow is an optional final phase of an Interaction.
    Interaction.Prompt
    A Interaction.Prompt represent a phase in an Interaction and is composed of a sequence of AudioItem and optionally a speech and/or a DTMF recognition configuration.
    Message
    A Message is a VoiceXmlOutputTurn that plays a sequence of AudioItem.
    Message.Builder
    Builder used to ease the creation of instances of Message.
    ObjectCall
    An ObjectCall is a VoiceXmlOutputTurn used to exploit platform-specific functionality.
    ObjectCall.Builder
    Builder used to ease the creation of instances of ObjectCall.
    ObjectCall.Parameter
    ObjectCall parameter, can be created with a string value, a JsonValue or an expression.
    OutputTurns
    Factory methods for various fluent builders for VoiceXmlOutputTurns.
    Recognition
    Base class of an interaction phase recognition configuration parts.
    Recording
    A Recording represents the description of a recording final phase of an interaction.
    Script
    A Script is a VoiceXmlOutputTurn that declares variables and/or executes a script.
    Script.Builder
    Builder used to ease the creation of instances of Script.
    SpeechRecognition
    A SpeechRecognition represents the speech part of an interaction phase recognition configuration.
    SubdialogueCall
    A SubdialogueCall is a VoiceXmlOutputTurn that invokes another external dialogue.
    SubdialogueCall.Builder
    Builder used to ease the creation of instances of SubdialogueCall .
    SubdialogueCall.Parameter
    SubdialogueCall parameter, can be created with a string value, a JsonValue or an expression.
    SupervisedTransfer
    A SupervisedTransfer is a Transfer where the outcome is monitored and where a transfer failure can be handled gracefully.
    SupervisedTransfer.Builder
    Builder used to ease the creation of instances of SupervisedTransfer
    Transfer
    This abstract class is the superclass of all classes representing a transfer to another entity.
    Transfer.Builder
    Base class for all Transfer builders.
    VoiceXmlOutputTurn
    This abstract class is the superclass of all classes representing a turn interpreted by the VoiceXML platform.
