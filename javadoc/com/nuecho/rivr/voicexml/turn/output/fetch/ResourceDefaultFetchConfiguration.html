Class ResourceDefaultFetchConfiguration

  • public class ResourceDefaultFetchConfiguration
extends java.lang.Object
    VoiceXML resource fetch configuration default properties: maxage , maxstale and fetchhint.

    This class defines default values for fetch properties and is used in DefaultFetchConfiguration which is used to generate VoiceXML global default properties for resource and document fetching.

    To specify the fetch properties on a per-resource basis, specify a FetchConfiguration on the required resource, for example AudioFile.setFetchConfiguration(FetchConfiguration).
