extends Object
implements DialogueChannelListener<I,O>
    Contains everything that is required for the dialogue to run in a DialogueServlet controller.

    NOTE: a Rivr Session can be linked with an HttpSession. This can be useful to maintain server stickyness in a clustered environment with load balancers. Web container generates JSESSIONID cookies for session tracking purpose but this information is also used by load balancer equipment between the HTTP user agent and the server to preserve server stickyness.
